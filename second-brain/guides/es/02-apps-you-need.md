# Aplicaciones que vas a necesitar

Buenas noticias: solo necesitás **una app principal** para armar tu Segundo Cerebro. Todo lo demás es opcional y se puede agregar después.

## Imprescindible

### 📝 Obsidian

Acá va a vivir tu Segundo Cerebro. Todo está acá.

| | |
|---|---|
| **Qué es** | App de toma de notas y gestión de conocimiento |
| **Por qué** | Archivos Markdown locales, wikilinks, plugins, vista de grafo |
| **Precio** | Gratis (app core). Sync y Publish son add-ons pagos. |
| **Plataformas** | macOS, Windows, Linux, iOS, Android |

**Descargalo:** [obsidian.md](https://obsidian.md)

Eso es todo. En serio. Podés parar acá y tener un Segundo Cerebro funcionando.

## Recomendados (Opcional)

Estas apps hacen más fácil capturar información desde distintas fuentes. Agregalas sobre la marcha — no instales todo de una.

### 📱 Obsidian Mobile

La app mobile te permite capturar ideas en movimiento. Disponible para iOS y Android.

- **iOS:** [App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442)
- **Android:** [Google Play](https://play.google.com/store/apps/details?id=md.obsidian)

### 🌐 Web Clipper — Obsidian

Extensión oficial del navegador para guardar páginas web, artículos y highlights directamente en tu vault.

- **Chrome:** [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/hoolezfiilemicimfjkbiedecpdjebih)
- **Firefox:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/obsidian-web-clipper/)

### ✅ Tasks (Plugin de Obsidian)

Manejá tareas y to-dos dentro de tus notas. No es una app separada — es un plugin que instalás dentro de Obsidian. Lo cubrimos en [05 — Plugins esenciales](./05-essential-plugins.md).

## Bueno tener (después)

Estos son para cuando tu Segundo Cerebro ya esté funcionando y quieras más potencia:

| App | Para qué | Cuándo agregar |
|-----|----------|----------------|
| **Readwise** | Sincronizar highlights de Kindle, artículos, podcasts | Cuando tengas un hábito de lectura |
| **Zotero** | Gestión de referencias académicas | Si hacés investigación |
| **Excalidraw** | Pensamiento visual y diagramas | Cuando pensás en imágenes |
| **Dataview** | Consultar tu vault como una base de datos | Cuando tengas 100+ notas |
| **Omnivore** | Guardar artículos para leer después | Cuando consumís mucho contenido web |

## Lo que NO necesitás

- ❌ Una suscripción paga a nada
- ❌ Cinco apps de notas distintas
- ❌ Un setup complejo con 20 herramientas
- ❌ Una compu o dispositivo nuevo
- ❌ Horas de configuración

## La Regla

> **Empezá con Obsidian. Agregá herramientas solo cuando sientas el dolor de no tenerlas.**

Si no estás chocando con un problema, no necesitás una herramienta para resolverlo.

## Checklist de descarga

- [ ] Obsidian descargado e instalado
- [ ] App mobile de Obsidian instalada (opcional)
- [ ] Extensión Web Clipper instalada (opcional)
