# Apps You'll Need

Good news: you only need **one main app** to build your Second Brain. Everything else is optional and can be added later.

## Required

### 📝 Obsidian

This is your Second Brain's home. Everything lives here.

| | |
|---|---|
| **What** | Note-taking and knowledge management app |
| **Why** | Local Markdown files, wikilinks, plugins, graph view |
| **Price** | Free (core app). Sync and Publish are paid add-ons. |
| **Platforms** | macOS, Windows, Linux, iOS, Android |

**Download:** [obsidian.md](https://obsidian.md)

That's it. Seriously. You can stop here and have a fully functional Second Brain.

## Recommended (Optional)

These apps make capturing information easier from different sources. Add them as you go — don't install everything at once.

### 📱 Obsidian Mobile

The mobile app lets you capture ideas on the go. Available for both iOS and Android.

- **iOS:** [App Store](https://apps.apple.com/us/app/obsidian-connected-notes/id1557175442)
- **Android:** [Google Play](https://play.google.com/store/apps/details?id=md.obsidian)

### 🌐 Web Clipper — Obsidian

Official browser extension to save web pages, articles, and highlights directly into your vault.

- **Chrome:** [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/hoolezfiilemicimfjkbiedecpdjebih)
- **Firefox:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/obsidian-web-clipper/)

### ✅ Tasks (Obsidian Plugin)

Manage tasks and to-dos inside your notes. Not a separate app — it's a plugin you'll install inside Obsidian. We'll cover this in [05 — Essential Plugins](./05-essential-plugins.md).

## Nice to Have (Later)

These are for when your Second Brain is established and you want more power:

| App | Purpose | When to add |
|-----|---------|-------------|
| **Readwise** | Sync highlights from Kindle, articles, podcasts | When you have a reading habit |
| **Zotero** | Academic reference management | If you do research |
| **Excalidraw** | Visual thinking and diagrams | When you think in pictures |
| **Dataview** | Query your vault like a database | When you have 100+ notes |
| **Omnivore** | Save articles for later reading | When you consume lots of web content |

## What You Do NOT Need

- ❌ A paid subscription to anything
- ❌ Five different note apps
- ❌ A complex setup with 20 tools
- ❌ A new computer or device
- ❌ Hours of configuration

## The Rule

> **Start with Obsidian. Add tools only when you feel the pain of not having them.**

If you're not running into a problem, you don't need a tool to solve it.
